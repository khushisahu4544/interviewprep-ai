#  InterviewPrep AI

A simple web application to practice interview skills.
## Features
- Category selection (JavaScript / HR)
- Multi-page navigation
- Clean UI design
- User input for answer
- next question navigation
- completion message after interview
##  Technologies Used
- HTML
- CSS
- JavaScript
##  Pages
- index.html → Home page(category selection)
- interview.html → Interview page(question+input)
## how its work
1. user selects a category(js/hr)
2. category is sorted using 'localStorage'
3. question are loaded dynamically based on selection
4. user answer each question
5. after completion-> success message is down

## future improvement
1. timer for each question
2. score system
3. progess tracking
4. live deployment


##  Purpose
This project helps users practice interview questions and improve confidence.
